# Survey Form Project

A clean, responsive, and functional Survey Form built using HTML, CSS, and JavaScript. This project was designed to demonstrate form structure, custom styling with modern fonts, and client-side form validation.

## 🚀 Features
*   **Fully Responsive:** Optimized for mobile, tablet, and desktop views using CSS Media Queries[cite: 3].
*   **Modern UI:** Styled with the 'Poppins' Google Font and a dark-themed color palette[cite: 3].
*   **Form Validation:** Includes custom JavaScript validation to ensure name and email fields are correctly filled before submission[cite: 2].
*   **Interactive Elements:** Features a variety of input types including dropdowns, radio buttons, checkboxes, and text areas[cite: 1].

## 🛠️ Technologies Used
*   **HTML5:** Structured the form semantic elements like `<header>`, `<form>`, and various `<input>` types[cite: 1].
*   **CSS3:** Custom styling including Flexbox for alignment and transitions for a smoother user experience[cite: 3].
*   **JavaScript (ES6):** Handled form submission events, prevented default behavior, and implemented email regex validation[cite: 2].

## 📸 Preview
The form includes:
1.  **Personal Info:** Name, Email, and Age[cite: 1].
2.  **Role Selection:** A dropdown menu for current professional status[cite: 1].
3.  **Feedback:** Radio buttons for recommendations and checkboxes for improvement suggestions[cite: 1].
4.  **Comments:** A vertical-resize textarea for additional user input[cite: 1, 3].
